### mock

运行 `npm run mock`

新建mock文件在 `./src/mock/data/mock.json` 目录

```json
{
  "code": 200,
  "data|1-9": [
    {
      "key|+1": 1,
      "title|1": ["肆无忌惮", "呵呵"],
      "content|1": [
        "角色精湛主题略荒诞",
        "理由太短 是让人不安",
        "疑信参半 却无比期盼",
        "你的惯犯 圆满",
        "别让纠缠 显得 孤单"
      ]
    }
  ]
}
```
接口地址为 `http:localhost:9001/mock`
